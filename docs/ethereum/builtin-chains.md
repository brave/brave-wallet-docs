# Built-in chains

Brave preloads the following chains by default.

| Chain ID    | Name                 | Native currency |
| ----------- | -------------------- | --------------- |
| 0x1         | Ethereum Mainnet     | ETH             |
| 0x3         | Ropsten Testnet      | ETH             |
| 0x5         | Rinkeby Testnet      | ETH             |
| 0x2a        | Kovan Testnet        | ETH             |
